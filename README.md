# Modelling Student Academic Success

Predicting student outcomes with a knowledge graph approach that integrates socioeconomic, behavioural, and educational factors from the Student Alcohol Consumption dataset.  
This project combines ontology construction, knowledge graph embeddings, and machine learning models to analyse and predict academic performance.

## Overview
This project builds an OWL/RDF knowledge graph from the UCI Student Alcohol Consumption dataset and uses it to model student academic success.  
The workflow includes ontology creation, graph validation, embedding generation with PyKEEN, and downstream regression and classification experiments for performance and dropout-related prediction tasks.

## Contents
- `student_performance.ipynb` - full implementation, experiments, and analysis.  
- `docs/` - includes the project report and presentation.
